

SNSI Web Platform

## Tech Stack
- Frontend: React + Leaflet + Tailwind CSS
- Backend: FastAPI + PostgreSQL + PostGIS

## Setup
1. **Database**: Follow the [Database Setup Guide](DATABASE_SETUP.md) to initialize your local data.
2. **Backend**: Navigate to `/backend`, install dependencies, and run `uvicorn app.main:app --reload`.
3. **Frontend**: Navigate to `/frontend`, run `npm install` and `npm start`.
